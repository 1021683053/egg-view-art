## Install

```bash
$ npm i egg-view-art --save
```

## Usage

```js
// {app_root}/config/plugin.js
exports.art = {
  enable: true,
  package: 'egg-view-art',
};
```

## Configuration

```js
// {app_root}/config/config.default.js
exports.art = {
};
```

see [config/config.default.js](config/config.default.js) for more detail.
